# 🩸 Pashtun Blood Network Lahore

> **One Blood • One Life • Humanity Before All**

Pashtun Blood Network Lahore (PBN) is a volunteer-based humanitarian initiative dedicated to connecting blood donors with patients in need. Our mission is to save lives by promoting voluntary blood donation and providing timely assistance during medical emergencies.

## 🌟 Mission

- Save lives through voluntary blood donation.
- Connect blood donors with patients quickly.
- Promote awareness about blood donation.
- Serve humanity without discrimination.

## 🚑 Features

- Blood Request
- Donor Registration
- Emergency Blood Support
- Donation Information
- Contact Information
- Mobile-Friendly Website
